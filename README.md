# Simple File compressor
- Frequency analyzer classical method is a program that counts the frequency of characters with in a file using basic/foundational logic: dynamic array of characters, with a reference array that checks if the loop already visited the same character or not to ensure it doesn't count for duplicate values different times. 

- Frequency analyzer program uses advanced concepts like vectors for dynamic structure, lambda sorting function, and range-based for loop. It also uses a tie breaker for same number of frequency, sorting characters in ascending order.
